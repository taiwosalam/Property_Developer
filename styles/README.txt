# Styles Folder

This folder contains all the styling resources for the project, including global CSS files, CSS modules, and Sass files. It is used to define and organize the visual appearance of the application.

## Example Folder Structure:
styles/
├── globals.css        # Global styles applied to the entire application
├── Home.module.css    # CSS module for the Home page
└── variables.scss     # Sass variables and mixins
